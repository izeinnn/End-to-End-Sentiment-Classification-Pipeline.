Pipeline Stage Walkthrough

We load the classic IMDb dataset using Hugging Face datasets module and downsample it to 2,000 train and 500 test samples to keep the training runtime predictable and student-friendly.

from datasets import load_dataset raw_datasets = load_dataset("imdb") train_dataset = raw_datasets["train"].shuffle(seed=42).select(range(2000))

Texts are tokenized using the pretrained DistilBERT tokenizer, truncated to 256 tokens, padded, and then transformed into PyTorch format with only tensor-compatible columns remaining.

def tokenize_function(examples): return tokenizer(examples["text"], truncation=True, padding="max_length", max_length=256) tokenized_train = train_dataset.map(tokenize_function, batched=True) tokenized_train.set_format("torch")

Training runs on PyTorch native DataLoader loaders utilizing an AdamW optimizer, a linear scheduler, and gradients computing on raw tensors.

optimizer = AdamW(model.parameters(), lr=2e-5) for epoch in range(EPOCHS): for batch in train_dataloader: outputs = model(**batch) loss = outputs.loss loss.backward() optimizer.step() optimizer.zero_grad()

We freeze the weights by invoking model.eval() and execute forward inferences on the test set dataloader, measuring accuracy via Hugging Face's evaluate module.

metric = evaluate.load("accuracy") model.eval() for batch in test_dataloader: with torch.no_grad(): outputs = model(**batch) predictions = torch.argmax(outputs.logits, dim=-1) metric.add_batch(predictions=predictions, references=batch["labels"])

In production, fine-tuned serialization weights are loaded back, text runs through a softmax layer to extract classification probabilities, and the highest scoring index determines target sentiment.

probabilities = torch.nn.functional.softmax(outputs.logits, dim=-1) sentiment = "Positive" if torch.argmax(probabilities) == 1 else "Negative"
